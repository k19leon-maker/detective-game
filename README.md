# Detective Telegram Mini App

Базовый каркас Telegram Mini App для детективной игры.

## Что уже есть

- React + Vite + TypeScript
- Подключение Telegram WebApp SDK
- Главный экран
- Экран дела
- Экран улик
- Экран персонажей
- Заметки игрока
- Нижнее меню
- Моковые данные для будущего сценария

## Как запустить локально

```bash
npm install
npm run dev
```

После запуска Vite покажет локальный адрес. Для Telegram Mini App понадобится публичный HTTPS URL.

## Как быстро протестировать в Telegram

1. Залить проект на GitHub.
2. Подключить к Vercel.
3. Получить HTTPS-ссылку вида `https://project-name.vercel.app`.
4. Открыть BotFather в Telegram.
5. Создать бота через `/newbot`.
6. Для мини-приложения использовать `/newapp` или добавить Web App кнопку в меню бота.
7. Вставить ссылку Vercel как URL мини-приложения.
